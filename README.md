## Getting Started

### Prerequisites

- NPM

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:renatozr/expo-memory-game.git
   ```
2. Enter the directory
   ```sh
   cd expo-memory-game
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run the project
   ```sh
   npm start
   ```
